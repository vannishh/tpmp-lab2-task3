# Программа учета успеваемости студентов

## Overview
Программа для управления данными о студентах и их успеваемости. Реализует следующие функции:
- Создание массива из 7 записей о студентах (ввод с клавиатуры)
- Вычисление среднего балла каждого студента
- Сортировка записей по убыванию среднего балла
- Вывод студентов, имеющих оценки только 4 и 5
- Удаление студента с минимальным средним баллом

## Modules
Данный репозиторий (`tpmp-lab2-task3`) представляет собой модуль (submodule) лабораторной работы №2.

### Сборка проекта
Сборка проекта осуществляется с помощью утилиты `make`. Для компиляции используется компилятор GCC с флагами:
- `-Wall` - включение всех предупреждений
- `-Wextra` - дополнительные предупреждения
- `-std=c99` - стандарт C99

### Структура сборки
```
tpmp-lab2-task3/
├── src/               # Исходные файлы
│   ├── main.c         # Главный модуль (клиент)
│   ├── student.h      # Определение структуры STUDENT (интерфейс)
│   ├── functions.h    # Заголовочные файлы функций (интерфейс)
│   └── functions.c    # Реализация функций (сервер)
├── build/             # Каталог для объектных файлов (создается при сборке)
├── docs/              # Документация
│   └── report.md      # Отчет о тестировании
├── Makefile           # Файл сборки
└── README.md          # Описание проекта
```

### Процесс сборки
1. При выполнении команды `make` создается директория `build/`
2. Исходные файлы из `src/` компилируются в объектные файлы в `build/`
3. Объектные файлы линкуются в исполняемый файл `student_program`
4. Для очистки используется команда `make clean`, которая удаляет директорию `build/` и исполняемый файл

### Автоматическая сборка
Проект настроен на автоматическую сборку с помощью GitHub Actions. Конфигурация находится в `.github/workflows/ci.yml`. При каждом push или pull request в репозиторий:
1. Запускается процесс сборки на Ubuntu
2. Выполняется команда `make`
3. Запускается скомпилированная программа

### Особенности реализации
1. **Создание массива записей**: функция `create_students()` осуществляет ввод данных для 7 студентов
2. **Вычисление среднего балла**: функция `calculate_averages()` рассчитывает среднее арифметическое оценок
3. **Сортировка**: функция `sort_by_average()` использует пузырьковую сортировку для упорядочивания по убыванию
4. **Фильтрация отличников**: функция `print_excellent_students()` выводит студентов с оценками только 4 и 5
5. **Удаление**: функция `remove_min_average_student()` удаляет студента с минимальным средним баллом
